# VigilentAids
AppForBlind People
VigilentAids is an Android app designed to support blind users by offering voice-guided navigation and real-time assistance. The app provides object recognition and reads text aloud, making everyday tasks easier and safer. VigilentAids also includes location-based alerts to connect users with emergency services when needed. With its customizable features, the app aims to improve the independence and quality of life for visually impaired individuals.
